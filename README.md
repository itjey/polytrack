# PolyTrack

A low-poly racing game inspired by TrackMania. Drive through loops, jumps, and high-speed tracks — every millisecond counts!

## 🎮 Play the Game

**[Play PolyTrack on GitHub Pages](https://itjey.github.io/polytrack/)**

## About

PolyTrack is an exhilarating driving game with a low-poly aesthetic featuring intense loops, jumps, and high-speed thrills where every millisecond is crucial. Build your own tracks or race on community-created ones.

This is version 0.4.1 of the open-source PolyTrack game, based on the original by [Kodub](https://kodub.itch.io/polytrack).

## Controls

- **Arrow Keys / WASD** — Drive
- **R** — Reset car
- **Escape** — Pause / Menu
